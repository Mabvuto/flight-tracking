# Flight Tracking

## Steps for run the app
 - Clone the project: git clone git@github.com:suca/flight-tracking.git
 - cd to project
 - npm install
 - bower install
 - grunt run 

## Steps for run the Server
 - cd to server
 - node index.js

#About the Endpoints
 - /api/flights
 - /api/flight/:id

 - /api/airports
 - /api/airport/:id
